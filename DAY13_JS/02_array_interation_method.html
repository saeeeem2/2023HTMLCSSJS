배열의 순환할 수 있는 구조를 이용하여 사용하는 메서드

  • array.reduce(callvack function(total, calue, index, self),[init value]); 연산의 누적 값을 리턴
  • array.reduceRight() : 배열의 우측에서 좌측방향으로 대입
  • array.every(callvack function(value, index, self)) : 콜백함수의 리턴 조건에 모든 배열의 값이 만족하면 true, 아니면 false
  • array.some(callvack function(value, index, self)) : 콜백함수의 리턴 조건에 배열의 값이 만족하면 true, 아니면 false (하나라도 만족하면)
  • array.indexOf('str',[start index]) : 해당 문자열이 존재하는 배열의 index를 리턴
  • array.lastIndexOf('str',[start index]) : 뒤에서부터 탐색
  • array.find(callback function(value,index,self)) : 검색 조건에 만족하는 첫 번째 배열 값 리턴
  • array.findIndex(callback function(value, index, self)) : 검색 조건에 만족하는 첫 번째 배열 인덱스 리턴
  • array.from('객체') : 순서가 있거나, 순호나 구조를 갖고 있는 객체를 배열로 변환
  • array.keys() : 배열이 사용 중인 key(식별자, index)를 검색하여 새로운 배열로 리턴 => Object는 적용 안 됨
  • array.entries() : index:value 의 구조값을 만들어 새로운 배열에 담아 리턴
  • array.includes('찾고싶은 값') : 배열에 찾는 값이 있으면 true, 없으면 false
