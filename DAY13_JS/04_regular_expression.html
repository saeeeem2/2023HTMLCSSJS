정규식(Refular Expression) : 문자열에서 특정 내용을 찾거나 대체, 발췌하는데 사용

  • regexp 단어로 많이 이용됨
  • 특정 데이터를 탐색, 확인, 존재여부 등을 쉽게 확인하기 위한 규칙
  • 웹 분야에서는 password 작성 규칙, 첨부파일규칙... 외에는 자주 쓰지 않음
  • 데이터 분석과 관련해서는 반드시 꼭 알고 있어야 함
  • 알고 있으면 편리한 기능이 있음, 다른 언어에서도 거의 비슷한 형식으로 활용
  • /regexp/i : (/ /) => 괄호,슬래시가 시작과 종료기호 regexp : pattern, i : flag
  • ("str").match(/regexp/flag) : str에서 매칭되는 항목들을 배열로 리턴
  • ("str").replace(/regexp/flag, "바꿀문자") : 정규 표현식에 매칭되는 항목을 바꿀 문자로 변경
  • ("str").split(/regexp/) : 정규 표현식에 매칭되는 항목으로 쪼개 배열로 리턴
  • (정규표현식).test('str') : 문자열이 정규표현식 규칙에 맞으면 true, 아니면 false
  • 정규표현식 플래그
  • i : 대소문자 구분 x
  • g : 전역검색
  • m : 행바꿈도 포함
  • a-zA-Z : 영어 알파벳 a-z : 소문자만 A-Z : 대문자만 ㄱ-ㅎ가-힣 : 한글 0-9 : 숫자
  • . : 모든 문자열(숫자, 한글, 영어, 특수기호, 공백 모두) 줄바꿈X
  • \d:숫자, \D:숫자 제외(숫자 아닌 것) \s:공백, \S:공백 제외(공백 아닌 것) \w:영숫자, \W:영숫자 제외(영숫자 아닌 것) \특수기호: ex. \!,\@... 해당 특수기호 허용
  • 검색패턴 : | OR a|b
  • [] : 괄호 안의 문자들 중 하나 or처리 묶음으로 보면 됨 /abc/ => "abc"를 포함하는 /[abc]/ => 'a'또는 'b'또는 'c'를 포함하는 [다-바] => 다 라 마 바 중 하나를 포함하는
  • [^문자] : 괄호 안의 문자를 제외한 것 [^abcd] : a,b,c,d, 4개의 문자를 제외
  • ^문자열$ : 시작(^)과 끝($) 표시
