가장 많이 다루는 데이터 타입 : String

  • java에서의 String 메서드와 거의 유사함
  • 문자열 데이터는 index(0부터 시작)
  • 문자열 값 내에서 개행(
    )하는 것을 추천하지 않음
  • 객체이므로 new String() 객체타입으로 선언 가능하나 추천하진 않음
  • length : 문자열의 총 길이
  • 추출관련 메서드 : substring(), substr(), slice()
  • 문자 대체 관련 메서드 : replace(ond,new) => 첫 매칭되는 문자만 가능/대소문자 구분
  • trim() : 양쪽 공백 제거
  • padStrat(길이, 채울문자) / padEnd(길이, 채울문자) ex. 010-********
  • chaAt(index) / indexOf()
  • startWith("str") / endWith("str") 해당 문자열 앞/ 뒤에 str이 존재하는지 확인하는 메서드 return true / false
