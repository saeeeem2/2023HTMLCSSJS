데이터 타입을 원하는 형식으로 변경(형변환, casting)

  • 문자 => 숫자 : Number("String number value"),parseInt("1"), parseFloat("0.5")
  • 숫자 => 문자 : 123.toString(), (123.12).toString()
  • 문자 => boolean : Boolean("true")
  • boolean => string : String(true)
  • number => boolean : Boolean(number value), Boolean(1) => true Boolean(0) => false
  • boolean => number : Number(true), Number(false)
