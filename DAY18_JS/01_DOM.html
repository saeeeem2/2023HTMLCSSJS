DOM(Document Object Model) : html(문서) 객체 모델

  • html하위 태그는 각각의 기능(method), 속성(properties)를 갖고 있는 문서 객체
  • element 가져오기 : tag element 를 가져오는 메서드 byId, byClassName, byTagName
  • document.getElementById('idName') : 단일 엘리먼트를 리턴
  • document.getElementByClassName('className') : 복수 엘리먼트를 리턴 => HTMLCollection타입으로 리턴
  • document.getElementByTagName('TagName') : 복수 엘리먼트를 리턴 => HTMLCollection타입으로 리턴
  • 복수 엘리먼트로 가져오기 되면 index를 선택해야 사용할 수 있음
