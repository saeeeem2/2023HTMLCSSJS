Map : key / value형태의 데이터를 한 개의 배열로 만들어서 다시 배열에 저장하는 형태

  • 선언 및 포기화 : new Map(), new Map([['key','value'],['key','value'],['key','value']...]);
  • 기본적으로 중첩 구조를 가지고 있기 때문에 key중복 확인 필수
  • key를 식별자로 사용하는 get(keyName), set(keyName, value)메서드 제공
  • 2차원 배열 형태와 비슷한 구조
  • key타입은 문자로 하는 것을 원칙으로 함 => key타입이 달라지면 관리가 어려워지기 때문
  • get(keyName) : key에 해당하는 value값을 리턴
  • set(keyName, value) : key에 해당하는 value를 변경
  • delete(key) / has() / forEach() / keys() / values() / entrise() / size / clear 등 set에서 쓰는 기능들은 모두 동일하게 제공(add는 없음, set으로 추가)
