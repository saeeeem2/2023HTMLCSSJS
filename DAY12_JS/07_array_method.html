배열관련 메서드 : 유사배열에서는 사용제한이 되는 메서드도 있음

  • array.push(value) : 배열의 마지막에 value를 추가하여 배열의 길이를 리턴 => array[array.length]=value; 같은 의미 리턴 : 배열의 길이를 리턴
  • pop() : 배열의 마지막 요소를 제거 리턴 : 배열의 마지막 요소를 리턴
  • shift() : 배열의 첫 요소를 제거 후 값을 리턴 => pop()과 비슷, 위치가 반대
  • unshift(value) : 배열의 첫 요소로 추가 => 길이를 리턴
  • join(value) : 배열 원소 사이에 문자를 삽입하여 새로운 문자열을 생성 리턴 : 문자열
