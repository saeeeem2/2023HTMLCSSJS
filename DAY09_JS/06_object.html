객체(object) : 식별할 수 있는 것을 상세 특징으로 표현하려는 형식

  • const로 선언하는 것이 일반적 : 객체의 구조를 지속적으로 유지하고 확보하기 위함
  • { } 속성명 : 값, 속성명 : 값, 속성명 : 값...
  • 속성명 : key, 값 : value
  • 같은 object내에 key(tnrjtdaud)는 중복 불가능(고유키)
  • 객체 내부에서 속성값으로 사용할 수 있는 데이터의 타입은 제한이 없다
  • 객체의 속성값으로 익명함수를 정의 => 그 함수를 메서드라고 부른다. 메서드의 이름은 속성명이 됨
