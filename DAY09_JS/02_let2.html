자바스크립트의 변수 사용

  • 값을 담고 있는 메모리 공간의 이름
  • 자바스크립트에서 변수는 그냥 선언하는 역할만 함
  • var, let(ES6부터는 let을 사용하는 것을 적극 권장)
  • 자바와 달리 변수에 자료형을 표시하지 않음
  • let으로 선언하면 어떤 값이든 다 담을 수 있음
  • var name='hong'; let name='hong'; name='kim'; //재할당은 가능
  • 지역변수 스코프 : {}
  • 스코프(scope: 사용범위) { }에서만 사용가능 스코프를 벗어나면 재선언 가능
  • 선언이후 항상 고정값을 가지는 경우 사용
  • 고정된 값을 사용하는 규칙, 참조, 룰, 객체, 엘리먼트 정의시 사용
