변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(notㅁ Number : 숫자가 아님)
  • 문자 : String= "", '',`` 감싼 값들
  • 불리언 : Boolean => true / false
  • 객체 : Object => 1개 이상의 데이터가 특정한 구조를 가지고 있는 형태 {키 : 값}({ } : 아무것도 없는 사실도 1개의 객체로 취급)
  • 배열 : Array => [1,2,3,...] 형태의 데이터 집합 (순서, 길이가 있음)
  • 알수없는 값 : null => 뭔가 있는 것 같긴한데 파악되지는 않은 값
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터의 타입을 확인하는 명령어 : typeof 변수

변수의 관리

  1. 기본자료형 : 값이 메모리의 주소값이 가르키는 곳에 직접 할당되는 값 number, boolean, string(참조형이지만 사용편의상 기본자료형과 같이 사용)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당 string, object, array, function, null, undefined
  3. == 같다(값만 비교) / !=같지 않다 ===같다(값과 타입이 같은지 비교) !==같지 않다(값과 타입이 같지 않은지 비교)
