함수(function) : 특정 기능을 수행하기 위해 작성한 코드(호출되어야지만 실행) 자바에서는 클래스안에서 메서드가 생성 자바스크립트는 어디서든 function생성 가능

  • function이라는 키워드를 사용하여 정의
  • function 함수명(파라미터){ 로 작성; }
  • 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) : 선택적으로 사용(없애도 ()괄호는 있어야함)
  • 자바스크립트에서 파라미터는 타입이 자유도가 높아서 다양한 혀애로 사용가능
  • return 함수종료역할과 동시에 특정 결과값을 호출한 위치로 변환
